# Icons Directory

## Required Icons

Pour que l'extension fonctionne correctement, ajoutez les icônes suivantes dans ce dossier :

- `icon16.png` (16x16 pixels)
- `icon32.png` (32x32 pixels)
- `icon48.png` (48x48 pixels)
- `icon128.png` (128x128 pixels)

## Recommandations

- **Format** : PNG avec fond transparent
- **Design** : Logo simple, icône de loupe ou checklist
- **Couleurs** : Indigo (#6366f1) pour correspondre au thème de l'extension

## Outils pour Créer des Icônes

- [Canva](https://www.canva.com/) - Création graphique simple
- [Figma](https://www.figma.com/) - Design professionnel
- [IconKitchen](https://icon.kitchen/) - Générateur d'icônes
- [Flaticon](https://www.flaticon.com/) - Bibliothèque d'icônes gratuites

## Note Temporaire

L'extension fonctionnera sans ces icônes, mais Chrome affichera une icône par défaut.
Pour une expérience professionnelle, ajoutez vos icônes personnalisées.
